Installation guide of my dotfiles (for MacOs)

1. Boostrap: Using the dotfile script, I install on a mac xcode, homebrew, and git.
2. I can now use git. I clone my repo in ./HOME .
3. I can now change the defaults config of the mac automaticaly and install everythong with brew from my brewfile.
4. Lastly, I install eveything else: 
    - MacOs Default Settings (https://macos-defaults.com/fr/)
    - zsh (default terminal / iterm2) and Oh My Zsh! (https://www.youtube.com/watch?v=MSPu-lYF-A8)
    - font : Iosevska (https://github.com/be5invis/Iosevka)
    - Neovim and pluggins (https://neovim.io/doc/user/lsp.html / https://github.com/hrsh7th/nvim-cmp)
    - vscode and some plugins
    - tmux
    - Yabai (i3 like window manager) (https://cbrgm.net/post/2021-05-5-setup-macos/)
    - fzf ()
